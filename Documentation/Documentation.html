What is Bitkal?

Bitkal is a programming language made for fun... nothing to special about it ._.

but here is how to use it!

Compiling Bitkal

To compile bitkal simply run the Compiler.bat wich should compile Bitkal for you

Run Command

The run command just makes it that instead of having to write all your commands in the bitkal terminal

you can just run files with bitkal code inside of them

For example make a file named whatever, in this example its named project, with the file extension .bk

Afterwards we can write are code in it

And then we can run this file by just writing in the terminal the command run("your_files_name.bk")

note: this wont work if shell.exe and your project.bk are in different file directories
